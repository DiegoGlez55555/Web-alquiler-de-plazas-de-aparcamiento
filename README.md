# Web-alquiler-de-plazas-de-aparcamiento

Aplicación web llamada **AparcaConmigo** para conectar a personas que buscan alquilar plazas de aparcamiento temporalmente con propietarios de plazas en la ciudad.

## Archivos

- `index.html`: Interfaz principal con búsqueda, filtros, listado y detalles.
- `styles.css`: Estilos con una estética relajada, botones suaves y diseño responsive.
- `script.js`: Lógica de filtrado, selección de plaza y visualización de reseñas.
- `manifest.json`: Información básica para soporte PWA ligero y potencial app móvil.

## Cómo usar

1. Abrir `index.html` en el navegador.
2. Buscar por ciudad, filtrar por texto, horario o día.
3. Hacer clic en una plaza para ver detalles completos y reseñas.
4. Contactar al anfitrión para acordar el pago.

## Potencial móvil

La aplicación está preparada para funcionar como una web app y puede integrarse en un contenedor móvil o PWA con soporte adicional de service worker.
